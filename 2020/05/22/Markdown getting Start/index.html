What is MarkDown

Markdown is a lightweight “markup language”. The founder is John Gruber, which replaces typesetting with concise grammar. It is recently widely used by more and more knowledge works, writing enthusiasts, programmers or researchers. There are no more than ten commonly used markup symbols. Compared to the more complex HTML markup language, Markdown is very lightweight, and the learning cost does not need too much.
